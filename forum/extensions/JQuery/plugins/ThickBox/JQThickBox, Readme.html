JQThickBox v12-3.1

by Luke Scammell aka [-Stash-]

Dependencies

  1. JQuery extension v13-1.2.1 or higher needs to be installed and enabled first in order for this to work.
  2. Vanilla 1.1.2 or higher.

Now developers can integrate ThickBox (3.1) functionality into their extensions quickly and easily, without having to worry about conflicting with other extensions using older/different versions of ThickBox.

Now using ThickBox 3.1! I'm using Klaus Hartl's ThickBox 2.1 variant (as it is a bit better than the *ahem* vanilla ThickBox 2.1.1) updated to work with JQuery 1.1+ and packed using Dean Edwards' JS packer.

From the ThickBox homepage

ThickBox is a webpage UI dialog widget written in JavaScript on top of the jQuery library. Its function is to show a single image, multiple images, inline content, iframed content, or content served through AJAX in a hybrid modal.

EXTENSION INSTALLATION INSTRUCTIONS

Download and uncompress the extension files, then upload them to:

/path/to/vanilla/extensions/JQThickBox/

The path to this extension's default.php file should look like this:

/path/to/vanilla/extensions/JQThickBox/default.php

To enable the extension you must be logged in with an account that has Administrator privileges. You can enable the extension in Vanilla by clicking on:

Settings > Extensions

Then tick the checkbox for "JQThickBox"
