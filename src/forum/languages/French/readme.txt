========================================
INSTRUCTIONS D'INSTALLATION DE LA LANGUE
========================================

Maintenant que vous avez votre paquet langage, vous devez le placer dans le dossier
des langues de Vanilla après avoir extrait l'archive .zip

/chemin/vers/vanilla/languages

Une fois que votre paquet langage est correctement installé, vous pouvez l'utiliser 
à partir du panneau de configuration "Language Management".

Notes: le fichier language.php contenu dans le dossier 'French' doit être déplacé 
dans le répertoire vanilla/conf de votre installation. Il définit les traductions
pour les extensions de Vanilla. Actuellement vous devez encore commenter
les définitions anglaises de ces extensions dans vanilla/extensions/<extension>/ 
pour que la traduction contenue dans vanilla/conf/language.php soit prise en charge.
