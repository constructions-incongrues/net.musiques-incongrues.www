Vanillacons

Version 1.2, By Jazzman (Maurice Krijtenberg) - May 1, 2006

A huge categorized library of insertable smilies for your posts

Installation

Extract the To-do list package to the Vanilla extension folder. Enable it by going to the settings page and selecting the checkbox for the To-do list extension. When the extension is enabled, check the Vanillacons settings and rebuild vanillacons.

NOTE: Make sure "smilies.js" and "smilies.php" are writeable!

Configuration

No configuration needed.
